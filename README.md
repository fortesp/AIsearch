# A.I Search Algorithms 

Contains a selection of the following algorithms:

- Uniform Cost
- Depth First Search
- Breath First Search
- [A*](https://en.wikipedia.org/wiki/A*_search_algorithm) (Breath First Search with a heuristic approach)
 
Example of problems to test:
- Missionaires and Cannibals
- Nickel and Dimme
- Buckets
- Eight Puzzle
- Specific graph path problem
